#  LoneForge — Official Website

Welcome to the repository for the **official LoneForge website**, the public-facing hub for the Solo RPG Engine project.

This site serves as a **showcase**, **documentation portal**, and **entry point** for the main LoneForge engine, which is currently under active development.

---

##  Project Status

The website is a work in progress and will continue to evolve as LoneForge grows.  
New sections, illustrations, and features will be added over time.

The **Tool (WIP)** page currently links to the main engine repository, but it will eventually host a fully interactive browser-based interface.

---

##  Main Engine Repository

The core procedural engine powering LoneForge can be found here:

 **https://github.com/AcciaioZero/LoneForge---Solo-RPG-Engine**

This is where you’ll find:
- the engine source code  
- procedural generation logic  
- technical updates  
- future APIs and modular components  

---

##  Hosting

The website is deployed using **GitHub Pages** and updates automatically whenever changes are pushed to the main branch.
